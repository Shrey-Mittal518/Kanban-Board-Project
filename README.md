# Kanban-Board-Project
## Overview
This project is a simple Kanban board application for task management.  
It helps organize tasks into categories like *To Do*, *In Progress*, and *Done*.

## Features
- Create, edit, and delete tasks
- Drag and drop tasks between columns
- Responsive design for desktop and mobile
